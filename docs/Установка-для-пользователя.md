# Установка Аманат Расписание

## Вариант 1. Самый простой

1. Откройте страницу релизов:
   `https://github.com/Scrymez/rasp_shool/releases/latest`
2. В блоке `Assets` скачайте файл:
   `amanat-raspisanie-setup-<version>.exe`
3. Запустите скачанный `.exe` двойным кликом.
4. Нажмите `Далее`, выберите папку установки или оставьте путь по умолчанию.
5. Дождитесь завершения установки.
6. Запустите ярлык `Аманат Расписание` на рабочем столе.
7. Войдите с паролем `admin`.
8. Сразу смените пароль во вкладке `Система`.

## Вариант 2. Автоустановка из файлов проекта

1. Скачайте проект ZIP с GitHub.
2. Распакуйте ZIP в любую папку.
3. Откройте папку проекта.
4. Запустите двойным кликом файл:
   `install-windows.cmd`
5. Скрипт сам скачает последний `.exe` из GitHub Releases и запустит установку.

## Где хранится база

```text
%APPDATA%/Аманат Расписание/data/scheduler.db
```

## Как обновить приложение

1. Откройте приложение.
2. Нажмите `Обновления`.
3. Нажмите `Проверить`.
4. Если новая версия найдена, нажмите `Скачать`.
5. После скачивания нажмите `Установить`.

## Если автоустановка не запускается

1. Проверьте интернет.
2. Проверьте, что репозиторий GitHub публичный.
3. Откройте `https://github.com/Scrymez/rasp_shool/releases/latest`.
4. Скачайте `.exe` вручную из `Assets`.
